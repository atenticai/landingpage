นโยบายคุกกี้

มีผลบังคับใช้: 1 มกราคม 2568  |  ปรับปรุงล่าสุด: 1 มกราคม 2568

🍪 เว็บไซต์นี้ใช้คุกกี้เพื่อปรับปรุงประสบการณ์การใช้งานของท่าน นโยบายนี้อธิบายประเภทคุกกี้ที่เราใช้ วัตถุประสงค์ และวิธีที่ท่านสามารถจัดการคุกกี้เหล่านั้นได้ ตาม พ.ร.บ. คุ้มครองข้อมูลส่วนบุคคล พ.ศ. 2562 (PDPA) เราจะใช้คุกกี้ที่ไม่จำเป็นก็ต่อเมื่อได้รับความยินยอมจากท่านเท่านั้น

1. คุกกี้คืออะไร?

คุกกี้คือไฟล์ข้อมูลขนาดเล็กที่เว็บไซต์จัดเก็บไว้ในเบราว์เซอร์ของท่านเมื่อท่านเข้าเยี่ยมชม คุกกี้ช่วยให้เว็บไซต์จดจำการตั้งค่าของท่าน วิเคราะห์การใช้งาน และปรับปรุงประสบการณ์การใช้งานโดยรวม

2. ประเภทคุกกี้ที่เราใช้

ประเภท ชื่อคุกกี้ วัตถุประสงค์ อายุ สถานะ
จำเป็น (Necessary) pdpa, pdpa_exp บันทึกการตัดสินใจ PDPA ของท่าน 180 วัน จำเป็น
จำเป็น (Necessary) acp_cd_end จดจำสถานะ Countdown Timer ใน Session Session จำเป็น
วิเคราะห์ (Analytics) _ga, _gid, _gat Google Analytics — วิเคราะห์จำนวนผู้เข้าชมและพฤติกรรมการใช้งาน 2 ปี / 24 ชม. ต้องยินยอม
การตลาด (Marketing) _fbp, _fbc Facebook Pixel — ติดตามประสิทธิภาพโฆษณาและ Retargeting 90 วัน ต้องยินยอม
การตลาด (Marketing) GTM cookies Google Tag Manager — จัดการและส่ง Tag ต่างๆ ขึ้นอยู่กับ Tag ต้องยินยอม

3. คุกกี้จากบุคคลที่สาม

นอกจากคุกกี้ของเราเอง เว็บไซต์อาจใช้บริการจากบุคคลที่สามดังนี้:

  • Google Analytics (Google LLC): วิเคราะห์สถิติการเข้าชมเว็บไซต์ — นโยบายความเป็นส่วนตัวของ Google
  • Google Tag Manager (Google LLC): จัดการ Tag และ Tracking Scripts — นโยบายความเป็นส่วนตัวของ Google
  • Meta Pixel (Meta Platforms Inc.): วัดผลโฆษณา Facebook/Instagram — นโยบายความเป็นส่วนตัวของ Meta
  • Google Fonts (Google LLC): โหลด Font Sarabun และ Prompt — ไม่เก็บคุกกี้สำหรับ Tracking

4. วิธีจัดการคุกกี้

🔧 วิธีปฏิเสธหรือลบคุกกี้ผ่านเบราว์เซอร์

ท่านสามารถตั้งค่าเบราว์เซอร์ให้ปฏิเสธคุกกี้ทั้งหมด หรือแจ้งเตือนเมื่อมีการส่งคุกกี้ โปรดทราบว่าการปิดคุกกี้ทั้งหมดอาจกระทบการทำงานบางส่วนของเว็บไซต์

🌐 Chrome ตั้งค่า → ความเป็นส่วนตัว → คุกกี้
🦊 Firefox ตัวเลือก → ความเป็นส่วนตัว → คุกกี้
🧭 Safari การตั้งค่า → ความเป็นส่วนตัว → คุกกี้
🔵 Edge ตั้งค่า → ความเป็นส่วนตัว → คุกกี้

ท่านยังสามารถเปลี่ยนใจเกี่ยวกับ PDPA Consent ได้ตลอดเวลา โดยล้าง Local Storage ของเบราว์เซอร์แล้วรีโหลดหน้าเว็บ Banner ยินยอมจะแสดงขึ้นมาใหม่ให้ท่านเลือกอีกครั้ง

5. การอ้างอิง Opt-Out ของบุคคลที่สาม

  • Google Analytics Opt-out: tools.google.com/dlpage/gaoptout
  • Facebook Ad Preferences: facebook.com/adpreferences
  • Network Advertising Initiative: optout.networkadvertising.org

6. การเปลี่ยนแปลงนโยบาย

เราอาจปรับปรุงนโยบายนี้เพื่อสะท้อนการเปลี่ยนแปลงในการปฏิบัติหรือข้อบังคับที่เกี่ยวข้อง หากมีการเปลี่ยนแปลงสำคัญ เราจะแจ้งให้ทราบผ่าน Banner บนเว็บไซต์
